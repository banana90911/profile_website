Stock Price Prediction

LSTM vs CNN-LSTM Deep Learning Research

개요

S&P 500 주가 예측을 위해 LSTM과 CNN-LSTM 모델의 성능을 체계적으로 비교 분석한 연구 프로젝트입니다. CNN-LSTM이 Feature Extraction과 시계열 분석의 결합으로 우수한 성능을 보임을 입증했습니다.

기술 스택

Python TensorFlow/Keras CNN-LSTM LSTM Data Analysis

주요 성과

  • CNN-LSTM이 LSTM보다 RMSE 58% 개선 (28.69 → 12.16)
  • CNN-LSTM의 상관계수 +0.57 (LSTM -0.38 대비)
  • Feature 추출과 시계열 분석의 효과 검증
  • 논문 형식의 체계적 분석 및 결론 제시

PHAI Project

Personal Project — 약학 AI 에이전트 플랫폼

개요

일반인을 위한 맞춤형 약학 정보 AI 에이전트 플랫폼입니다. GPT-4o-mini와 LlamaIndex를 활용하여 멀티에이전트 아키텍처로 약물 정보, 복약 지도, DUR(의약품 안전사용 정보) 등을 통합 제공합니다. PIPA 규정을 준수하여 개인정보를 안전하게 보호합니다.

기술 스택

FastAPI LlamaIndex GPT-4o-mini ChromaDB Python

주요 특징

  • 멀티에이전트 워크플로우 (Classification → RAG → DUR → Review → Synthesize)
  • 식약처 공공 API 데이터 통합 (e약은요, DUR, 건강기능식품)
  • Presidio 기반 PII 마스킹으로 개인정보 보호
  • FastAPI 스트리밍 응답으로 실시간 사용자 경험
  • Stateless 아키텍처로 확장성 확보
